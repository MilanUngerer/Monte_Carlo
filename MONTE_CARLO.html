Oscilador armónico unidimesional con el Método de Monte Carlo¶

Modelo¶

Consideramos un oscilador armnico de una dimensin con un espectro de energa dado por:

$E_n = \hbar \omega (n+1/2)$

Para su resolucin se utilizar el proceso de Markov que considera que cada estado microscpico slo depende del ltimo estado y no de los visitados anteriormente. Adicionalmente, se tiene la Condicin de Balance Detallado (CBR) descrita a continuacin.

$W(q' \rightarrow q)P_{eq}(q')-W(q \rightarrow q')P_{eq}(q)=0$

A partir de esto, podemos utilizar el algoritmo de Metrpolis para aplicar el mtodo de Monte Carlo a nuestro sistema.

@jit
def E(it, beta, n, hw, cont):
    En = hw*(n + 0.5) #energia inicial
    E = np.zeros(it)
    E2 = np.zeros(it)
    Eprom = np.zeros(it)
    E2prom = np.zeros(it)
    Eprom2 = np.zeros(it)
    Sigma = np.zeros(it)
    narr = np.zeros(it)
    for i in range (it):
        nprima = n + np.random.choice([-1,1]) #estado n'
    
        if nprima < 0:
            nprima = n
        else:
            nprima = nprima
        
        Enp = hw*(nprima + 0.5) #energia para estado n'
        DE = Enp - En #delta de energia
        if DE <= 0:
            n = nprima
            En = Enp

        elif DE > 0:
            zeta = np.random.random_sample() #numero random entre 0,1
            alfa = zeta - np.exp(-beta*DE)
            if alfa >= 0:
                n = n
            elif alfa < 0:
                n = nprima
                En = Enp
    
        E[i] = En                       #energia
        E2[i] = E[i]**2                 #energia cuadrado
        narr[i] = n                     #nivel
        Eprom [i] = np.mean(E[:i])   #energia promedio
        E2prom [i] = np.mean(E2[:i])   #energia cuadrado promedio
    
    Eprom2[:] = Eprom[:]**2         #energia promedio hasta cada MCS al cuadrado
    Sigma[:] = E2prom[:]-Eprom2[:]  #fluctuacion
    
    return Eprom, Sigma

Comparación con valor "exacto"¶

Ahora compararemos nuestro resultado con el valor exacto dado por el ensemble cannico

$\langle\epsilon\rangle = \hbar\omega \left(\frac{1}{e^{\frac{\hbar\omega}{kT}}-1} + \frac{1}{2} \right)$

Podemos calcular adems,

$\langle E^2 \rangle - \langle E \rangle^2 = kT^2 C_v$

$\frac{C_v}{N} = k \left( \frac{\hbar \omega}{kT} \right)^2 \frac{e^{\hbar\omega/kT}}{\left( e^{\beta\hbar\omega} -1 \right)^2}$
